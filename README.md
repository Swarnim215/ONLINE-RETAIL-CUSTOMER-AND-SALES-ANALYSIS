# E-Commerce Customer Segmentation & Business Analytics Pipeline

An end-to-end data analytics, machine learning, and business intelligence pipeline that transforms raw e-commerce transactional data into actionable corporate strategies. This project combines robust data engineering, exploratory data analysis (EDA), automated RFM modeling, unsupervised machine learning (K-Means Clustering), and interactive Power BI dashboards to drive customer retention and maximize lifetime value (CLV).

---

## 📌 Project Overview

Using a historical dataset containing over a year of transactional records from an online retail store (UCI Machine Learning Repository), this system builds an enterprise-grade analytics pipeline. The core objective is to move from basic descriptive statistics to prescriptive corporate actions by segmenting the customer base according to behavioral patterns and risk metrics.

## 🚀 Key Features & Implementation Details

### 1. Data Engineering & Preprocessing (`Python`)

* **Data Sanitation:** Handled missing values by isolating non-registered users (`CustomerID`), systematically eliminated duplicate entries, and scrubbed transactions with invalid arrays (e.g., non-positive unit prices).
* **Transaction Controls:** Identified and isolated canceled orders (invoices beginning with "C") to preserve baseline revenue integrity.
* **Feature Production:** Engineered a core transactional `Revenue` metric:

$$\text{Revenue} = \text{Quantity} \times \text{UnitPrice}$$



### 2. Behavioral Feature Engineering (RFM Model)

Calculated behavioral profiles for every unique customer across three primary vectors:

* **Recency ($R$):** Days elapsed since the customer's last valid transaction relative to a pipeline snapshot date.
* **Frequency ($F$):** Total count of unique, non-canceled invoices generated by the customer.
* **Monetary ($M$):** Aggregated revenue contribution across the entire customer lifecycle.

### 3. Machine Learning Customer Clustering

* **Feature Normalization:** Applied `StandardScaler` to map the high-variance RFM distributions to a unified scale ($\mu = 0, \sigma = 1$), ensuring unbiased distance calculations.
* **Optimization:** Utilized the **Elbow Method** evaluating Within-Cluster Sum of Squares (WCSS) to mathematically determine the optimal cluster configuration ($K$).
* **Segmentation:** Trained a **K-Means Clustering** algorithm to group customers into clear strategic tiers: *High-Value*, *Medium-Value*, and *Low-Value* cohorts.
* **Predictive Churn Stratification:** Layered an algorithmic rule-matrix over the RFM values to proactively isolate customers demonstrating high attrition risks.

### 4. Enterprise Business Intelligence (`Power BI`)

Engineered a comprehensive `.pbix` reporting stack leveraging custom Data Analysis Expressions (DAX) and clean data models across three interactive surfaces:

* **Executive Sales Dashboard:** High-level operational metrics (Total Revenue, AOV, Monthly Growth Trends, Regional Distribution).
* **Customer Analytics Dashboard:** Granular distribution profiles, repeat purchase rates, and value segmentations.
* **Product Performance Dashboard:** Inventory velocity analysis, top revenue generators, and quantity distribution grids.

## 📊 Business Impact & Prescriptive Strategy

The pipeline automatically maps behavioral segments to dedicated multi-channel marketing actions:

* **Champions / High-Value:** Targeted with premium VIP loyalty programs, early product releases, and custom rewards.
* **Potential Loyalists / Medium-Value:** Injected into cross-selling and up-selling loops to systematically maximize share-of-wallet.
* **At-Risk / High Churn Risk:** Auto-targeted with proactive win-back discount structures and targeted re-engagement mailing sequences before customer churn occurs.
